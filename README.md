# TetrisOpeningSolver

An attempt to make a React app that allows for users to enter pieces on a tetris board and analyze the probability that the opening bag's order can support the structure.

## Purpose

Learning openings I realized that there were some I could not make with some opening bags, sometimes it is hard to find info on which bags are usable for certain openings and tools out there are hard to use.
